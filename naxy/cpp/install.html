How to Install


Windows Install Media

こちらから最新版Naxy.C++.Library.Setup.msiをダウンロードし実行してください

注意: Windows セキュリティー及びブラウザにダウンロードを止められる可能性がありますが、ウイルスではございません。不安な方はZipをご利用ください。

実行後、指示に従ってインストールを進めてください。必要空き容量は1MB未満です。

デフォルトパスは以下の通りです。

C:\Naxy\lib\cpp


Zip

最新版のZipをダウンロード・展開し、任意の場所に配置してください。
